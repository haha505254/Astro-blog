歡迎來到我的部落格！這是我使用 **Astro + MDX** 建立的第一篇文章。今天想和大家分享為什麼我決定開始寫部落格，以及這個部落格系統的強大功能。

## 為什麼開始寫部落格？

寫部落格對我來說有幾個重要的意義：

1. **記錄學習歷程** - 把學到的知識整理成文章，加深理解
2. **分享經驗** - 幫助其他遇到相同問題的開發者
3. **建立個人品牌** - 在技術社群中建立自己的聲音
4. **提升寫作能力** - 練習將複雜概念用簡單的方式表達

<Alert type="info">
  寫部落格不只是分享知識，更是一個自我成長的過程！
</Alert>

## 這個部落格的特色

### 1. 極致效能

使用 Astro 靜態生成，載入速度飛快：

- ⚡ 幾乎零 JavaScript
- 🚀 優化的圖片載入
- 📦 最小化的 bundle size

### 2. MDX 支援

可以在 Markdown 中使用 React 組件，讓文章更生動：

<Card title="互動式卡片範例">
  這是一個可以在文章中使用的 React 組件！你可以加入任何互動功能。
</Card>

### 3. 程式碼高亮

支援各種程式語言的語法高亮：

```javascript
// JavaScript 範例
function greet(name) {
  return `Hello, ${name}! 歡迎來到我的部落格！`;
}

console.log(greet('讀者'));
```

```css
/* CSS 範例 */
.blog-post {
  max-width: 800px;
  margin: 0 auto;
  padding: 2rem;
}
```

### 4. 深色模式

## 文章格式展示

### 文字格式

- **粗體文字** 使用 `**text**`
- *斜體文字* 使用 `*text*`
- ~~刪除線~~ 使用 `~~text~~`
- `行內程式碼` 使用 `` `code` ``

### 列表

**無序列表：**
- 項目一
- 項目二
  - 子項目 A
  - 子項目 B
- 項目三

**有序列表：**
1. 第一步
2. 第二步
3. 第三步
