歡迎來到我們全新的 Astro + MDX 部落格！這個部落格展示了結合 Astro 靜態網站生成器和 MDX（Markdown + JSX）的強大功能。

## 什麼是 MDX？

MDX 是一種將 JSX 組件嵌入到 Markdown 文件中的格式。這意味著你可以：

- 📝 使用熟悉的 Markdown 語法撰寫內容
- 🚀 嵌入 React 組件來增強互動性
- 🎨 建立動態和可重用的內容區塊

<Alert type="info">
  MDX 讓你的文件活起來！你可以在 Markdown 中直接使用 React 組件。
</Alert>

## 主要特色

### 1. 組件整合

你可以在 MDX 文件中直接導入和使用 React 組件：

<Card title="互動式卡片">
  這是一個在 MDX 中使用的自定義 React 組件。你可以傳遞 props 並建立互動式內容！
</Card>

### 2. 程式碼高亮

MDX 支援語法高亮的程式碼區塊：

```jsx
// React 組件範例
const WelcomeMessage = ({ name }) => {
  return (
    <div className="p-4 bg-blue-100 rounded-lg">
      <h3 className="text-xl font-bold">Hello, {name}!</h3>
      <p>歡迎來到 Astro + MDX 的世界！</p>
    </div>
  );
};
```

### 3. 表格支援

| 功能 | Markdown | MDX | 說明 |
|------|----------|-----|------|
| 文字格式 | ✅ | ✅ | 標題、粗體、斜體等 |
| 程式碼區塊 | ✅ | ✅ | 支援語法高亮 |
| React 組件 | ❌ | ✅ | MDX 的殺手級功能 |
| 互動性 | ❌ | ✅ | 可以加入動態內容 |

## 為什麼選擇 Astro？

Astro 是一個專為內容網站設計的現代靜態網站生成器：

1. **零 JavaScript 預設** - 只在需要時才載入 JavaScript
2. **內容優先** - 專為部落格和文檔網站優化
3. **框架無關** - 支援 React、Vue、Svelte 等多種框架
4. **極致效能** - 產生最小的 HTML 和 CSS

## 開始使用

要開始使用這個部落格模板，你可以：

1. 複製這個專案
2. 安裝相依套件：`npm install`
3. 啟動開發伺服器：`npm run dev`
4. 開始撰寫你的第一篇 MDX 文章！
